# ArticleWrite 写作空间 - Claude 协作规则

## 项目简介

这是一个用于文章写作和内容创作的工作空间，包含 Claude Code Skills 技能集合。

## 自动提交规则

**重要规则：静默自动提交到 GitHub**

- 在合适的时机自动提交并推送到 GitHub
- 无需询问用户或通知
- 提交时机包括：
  - 完成重要工作后
  - 创建或修改文档后
  - 记录讨论内容后
  - 创建新目录后
  - 移动文件到新目录后
  - 其他有意义的变更

## 目录管理规则

- 新的写作文档必须归属于某个已有目录，不得直接在根目录创建
- 每个文档都应该有清晰的分类归属
- 目录名称使用有意义的英文命名
- 创建新目录后自动提交目录结构
- 移动文件到新目录后提交目录变更

## 提交信息规范

- 使用清晰简洁的中文提交信息
- 提交格式：`<类型>: <简短描述>`
  - 文档: `docs: 更新CLAUDE.md`
  - 新建: `feat: 添加新文章草稿`
  - 修改: `edit: 修订文章内容`
  - 修复: `fix: 修正错误`

## 常用操作

```bash
# 查看状态
git status

# 查看日志
git log --oneline -10
```

本仓库包含多种 Claude 技能，用于辅助写作和文档处理：
