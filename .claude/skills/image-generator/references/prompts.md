# Prompt Writing Guide

Best practices and examples for writing effective image generation prompts.

## Prompt Structure

A good prompt follows this structure:
```
{Subject} + {Style} + {Details} + {Quality Modifiers}
```

## Key Elements

### 1. Subject (What)
The main thing you want to see.

**Bad**: "a dog"
**Good**: "A golden retriever puppy running through autumn leaves"

### 2. Style (How)
The artistic style or medium.

**Examples**:
- "photography, realistic, 8K"
- "digital painting, concept art"
- "oil painting, impressionist"
- "3D render, minimalist"
- "anime style, Studio Ghibli"

### 3. Details (Specifics)
Important visual details.

**Examples**:
- "warm sunset lighting"
- "cyberpunk city, neon lights"
- "minimalist composition, white background"
- "depth of field, blurred background"

### 4. Quality Modifiers (Technical)
Enhance the technical quality.

**Examples**:
- "high quality, detailed"
- "8K, ultra HD"
- "professional, masterpiece"
- "sharp focus, well-lit"

## Example Prompts

### Technology & Business

**Article cover (16:9)**:
```
Abstract three-layer evolution diagram showing API progression from text completion to chat bubbles to intelligent agent network, modern tech style, gradient blue and purple colors, professional presentation quality, clean background
```

**Tech illustration**:
```
Futuristic AI brain circuit board, glowing blue and purple, digital art, high tech, detailed, professional, white background
```

**Mountain landscape**:
```
Majestic snow-capped mountains at sunset, reflection in crystal clear lake, landscape photography, dramatic lighting, 8K, ultra detailed
```

**Forest scene**:

```
Professional portrait of a software engineer, modern office background, studio lighting, high quality photography, realistic
```

```
Elegant elven warrior with silver armor, forest background, fantasy art, detailed, cinematic lighting, epic
```

```
Abstract digital waves flowing from simple to complex, gradient blue to purple, modern art, clean, minimalist
```

## Negative Prompts (What to Avoid)

When using services that support negative prompts (Stability AI), specify what you DON'T want:

```
blurry, low quality, distorted, watermark, text, logo, bad anatomy, ugly, duplicate
```

## Tips for Better Results

1. **Be Specific**: More details = better results
2. **Order Matters**: Subject → Style → Details → Quality
3. **Use Keywords**: "high quality", "detailed", "professional"
4. **Avoid Ambiguity**: "a nice picture" → "sunset over ocean"
5. **Iterate**: Start simple, add details gradually
6. **Test First**: Use free service (Pollinations) to test prompts

---

## Common Mistakes

❌ **Too vague**: "make it look good"
✅ **Specific**: "modern minimalist design, clean lines, professional"

❌ **Contradictory**: "realistic photo, cartoon style"
✅ **Consistent**: "realistic photo, professional lighting"

❌ **Too long**: 500+ words of description
✅ **Concise**: 50-100 well-chosen words
